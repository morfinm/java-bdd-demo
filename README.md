# Java Selenium Cumber BDD Test Automation Framework 

This repository provides a baseline BDD test automation framework for the User Registration feature on https://practicesoftwaretesting.com/.

Framework Structure:
```\njava-bdd-demo/
  src/
    main/
      java/
        com/practicesoftwaretesting/
          pages/
        utils/
    test/
      java/
        com/practicesoftwaretesting/
          stepdefinitions/
          runners/
      resources/
        features/
````

Features: 
- user_registration.feature covers all acceptance criteria from US-001 user story.

## Running the Tests

To
 - run tests use maven command:
    mivn test

## Requirements 
- Java 11 or higher 
- Selenium 4.1 +: Chrome draiver
- Cumber 7.1 + - Junit 4.1
